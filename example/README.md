# 🌧️ 워밍업 예제 — "하늘에 먹구름이 많아지면 뭐가 생각나?"

Colab **한국어→영어 번역 미니 트랜스포머** 과제를 시작하기 전에, 트랜스포머의 핵심 직관을
**한 문장**으로 먼저 익히는 초간단 예제입니다. 아주 작은 "질문 → 답" 데이터로 미니 트랜스포머를
직접 학습시키고, **어텐션·순차 생성·크로스 어텐션**을 눈으로 확인합니다.

> 🎯 핵심 훅: 답 "**비**"를 만들려면 모델이 질문의 "**먹구름**"에 **주목**해야 합니다.
> 이게 바로 어텐션이 하는 일입니다.

---

## 무엇을 보여주나

`python demo.py` 를 실행하면 세 가지가 출력됩니다.

1. **순차 생성(마스킹)** — 답을 `<sos>`부터 한 단어씩, **앞말만 보고** 고르는 과정
   ```
   '<sos>'          →  다음 단어: 비가
   '비가'           →  다음 단어: 올
   '비가 올'        →  다음 단어: 것
   '비가 올 것'     →  다음 단어: 같아
   답  : 비가 올 것 같아
   ```
2. **크로스 어텐션 히트맵** (`attention_heatmap.png`) — 답의 각 단어가 질문의 어느 단어를 봤는지.
   답의 첫 단어가 **먹구름이**에 가장 밝게 쏠립니다.
3. **대조 예시** — 문장 구조는 같아도 키워드가 바뀌면 주목 대상과 답이 함께 바뀝니다.
   ```
   먹구름 → 비가 올 것 같아
   별     → 밤이 깊었나 봐
   해     → 아침이 밝았구나
   ```

## 트랜스포머 개념 ↔ 이 예제

| 개념 | 이 예제에서는 |
|------|--------------|
| 토큰화 | `하늘에 / 먹구름이 / 보이면 / 뭐가 / 생각나` 로 자르기 |
| 임베딩 | 각 단어를 의미 좌표(벡터)로 변환 |
| 위치 인코딩 | 순서 정보를 sin/cos 파도무늬로 주입 |
| **셀프 어텐션** | 답을 내려고 **먹구름이**(답을 가르는 유일한 단어)에 밑줄(높은 가중치) |
| 멀티헤드 | 여러 관점(무엇/추세/질문형)으로 동시에 주목 |
| 잔차+정규화+FFN | 모은 문맥을 각 단어가 혼자 더 곱씹기 |
| **마스킹 디코딩** | 답을 한 단어씩, 미래를 못 보고 생성 |
| **크로스 어텐션** | 답을 쓰는 내내 질문(원문)을 곁눈질 |

> Colab 번역 과제로 가면 **질문→답**이 **한국어→영어**로 바뀔 뿐, 구조는 똑같습니다.

## 실행 방법

### 1) 가상환경 만들고 의존성 설치

**Windows (PowerShell)**
```powershell
cd example
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

**macOS / Linux**
```bash
cd example
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

> `torch` 설치가 오래 걸리면 CPU 전용 경량 휠을 쓰세요:
> `pip install -r requirements.txt --extra-index-url https://download.pytorch.org/whl/cpu`

### 2) 실행

```bash
python demo.py
```

CPU에서 수 초 안에 끝납니다(GPU 불필요). 실행 후 같은 폴더에 `attention_heatmap.png` 가 생깁니다.

## ⚠️ 이 예제의 한계 (일부러 작게 만든 것)

- 학습 데이터가 **6쌍뿐**이라 모델은 사실상 **암기**합니다. 목적은 성능이 아니라 **구조 이해**입니다.
- "일반화"를 보려면 데이터가 훨씬 많아야 합니다 — 그건 Colab 본 과제에서 다룹니다.
- 즉, 이 예제는 *"트랜스포머가 어떻게 답을 만드는지"* 를 눈으로 느끼는 **워밍업**입니다.
